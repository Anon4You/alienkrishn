# alienkrishn
A custom termux apt repository maintained by me 

## add key 

```
curl -sSL https://github.com/Anon4You/alien.repo/raw/main/alienkrishn.key | apt-key add
```

## add source list

```
echo "deb [trusted=yes arch=all] https://anon4you.github.io/alien.repo anon main" > $PREFIX/etc/apt/sources.list.d/alienkrishn.list
```
